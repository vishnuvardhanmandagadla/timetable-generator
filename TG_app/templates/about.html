Project Overview

Our current project involves developing a comprehensive timetable generator using Django. This project includes:

  • Generating timetables for multiple academic years and departments.
  • Scheduling labs and courses without faculty collisions.
  • Ensuring each course is adequately covered within the timetable.
  • Implementing a user-friendly interface for timetable generation and management.

Technologies Used

  • Django
  • Python
  • HTML, CSS
  • JavaScript
  • CSV data manipulation
